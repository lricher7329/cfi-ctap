## Pricing Structure

### Enterprise Edition vs. Community Edition

| Feature | Community (Free) | Enterprise |
|---------|-----------------|------------|
| Core LIMS functionality | Yes | Yes |
| User limit | Unlimited | Unlimited |
| Premium features | No | Yes |
| Professional support | No | Yes |
| Custom development | No | Available |
| SLA | No | Yes |

### Enterprise License Pricing Factors

OpenSpecimen pricing is customized based on:
- Number of specimens managed
- Number of sites/institutes
- Required premium features
- Support level required
- Implementation complexity

#### Estimated Pricing (CAD)

| Component | Estimated Annual Cost | 5-Year Total |
|-----------|----------------------|--------------|
| Enterprise License (base) | $25,000 - $40,000 | $125,000 - $200,000 |
| Premium support package | $10,000 - $15,000 | $50,000 - $75,000 |

**Note:** Washington University at St. Louis supports 50+ research groups, 80+ active users, and ~2 million specimens on their enterprise deployment.
