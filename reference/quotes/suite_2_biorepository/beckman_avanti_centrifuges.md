## Recommended Systems

### Primary: Beckman Coulter Avanti J-26S XP

**Manufacturer:** Beckman Coulter
**Website:** [beckman.com/avanti-j-26s-series](https://www.beckman.com/centrifuges/high-speed/avanti-j-26s-series)

#### Key Features
- Maximum RCF: 82,000 × g
- Capacity: Up to 6 liters
- Temperature range: -10°C to +40°C
- BioSafe containment features
- Automated Rotor Identification
- Full containment barrier

#### Biocontainment Specifications
- BioSafe aerosol containment
- HarvestLine System Liners for reduced sample handling
- Sealed rotor options available
- Suitable for BSL-2 applications
